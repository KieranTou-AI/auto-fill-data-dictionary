# 数据字典填写规范

在数据字典中定义数据类型和操作类型时，采用以下缩写。

## 数据类型（DT）

| 类型 | 缩写 | 说明 |
|---|---|---|
| 字符 | C | 字符串类型，string |
| 数字 | N | 1. 数值类型，包括 int、float、double、long<br>2. 其中 float 型数据，保留小数点后 2 位 |
| 日期 | D | 日期类型，无时间 |
| 日期时间 | DT | 1. 日期时间类型，包含日期和时间<br>2. 对于过长的日期时间，显示英文缩写单位 YMD hms，算上数值仅保留 6 位数值，由大到小保留，鼠标移动上显示中文全部信息 |
| 时间 | T | 仅包含时间，无日期 |
| 空 | E | 空值，空状态 |
| 按钮 | B | 是可操作按钮类型 |

## 操作类型（OT）

| 类型 | 缩写 | 说明 |
|---|---|---|
| 必填项 | M | 是必须配置的配置项，表单需进行标识必须配置的标记，并做非空校验 |
| 选填项 | O | 可填，可不填，无标识，不必做非空校验 |
| 可修改 | A | 配置保存后，可再次进行修改 |
| 不可修改 | FA | 配置保存后，就不可进行修改 |
| 显示项 | D | 一般为监控数据或自动显示的数据，仅显示，不是配置的 |
| 可点击文本 | TC | 文本可点击或对应的文本存在可点击的图标 |
| 按钮长显 | BL | 单击操作按钮，长期显示，点击后判断是否有效，具体在对应的业务中给出按钮含义 |
| 按钮置灰 | BG | 单击操作按钮，不可点击时，置灰显示，具体在对应的业务中给出按钮含义，并给出不可点击的原因，鼠标移动悬浮提示 |
| 按钮消失 | BD | 单击操作按钮，不可点击时，按钮消失，具体在对应的业务中给出按钮含义 |
| 文本输入框 | TB | 文本的输入，配合其他的操作类型呈现 |
| 隐藏项 | H | 后台数据，不呈现给用户的数据 |
| 下拉单选框 | DRB | 下拉单选框，仅可选一个对象 |
| 下拉多选框 | DMB | 多选，可选择多个对象 |
| 单选框 | R | 单选框，仅可选一个对象 |
| 开关 | S | 开关，可切换开启或关闭 |
